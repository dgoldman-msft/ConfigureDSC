# InstallPreReqs

## SYNOPSIS
Install system prerequisites

## SYNTAX

```
InstallPreReqs [-Exchange] [<CommonParameters>]
```

## DESCRIPTION
This will install all PowerShell prerequisite as well as other workloads

## EXAMPLES

### EXAMPLE 1
```
InstallPreReqs
```

### EXAMPLE 2
```
InstallPreReqs -Exchange
```
